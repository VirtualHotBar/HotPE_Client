此仓库为HotPE的聚合客户端，技术栈:Electron,React,TypeScript,Semi Design

![](https://my-img.cc/i/2023/02/18/63f022cfd37e3.png)

# 使用

### 安装依赖
`npm install`

### 开发
启动主线程开发：
`dev-main.bat`

启动渲染线程开发：
`dev-render.bat`

启动electron开发：
`dev-electron.bat`

### 构建
构建主线程：
`build-main.bat`

构建渲染程：
`build-render.bat`

构建electron应用：
`build-electron.bat`
