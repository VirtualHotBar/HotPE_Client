# HotPE Client

# 介绍

此仓库为HotPE的聚合客户端，技术栈:Electron,React,TypeScript,Semi Design

![](https://p2.myzwq.com/i/2022/11/19/637864867904d.png)

# 使用

### 安装依赖
`npm install`

### 开发
启动主线程开发：
`dev-main.bat`

启动渲染线程开发：
`dev-render.bat`

启动electron开发：
`dev-electron.bat`

### 构建
构建主线程：
`build-main.bat`

构建渲染程：
`build-render.bat`

构建electron应用：
`build-electron.bat`
